esta es la disrubiiconde las capetas

-archivo = ejecutar.bat: este archivo ejecuta el script.ps1

-capeta = view: esta capeta contiene un archivo index.xaml

crear.ps1: este recivira los praramtro de la vista para utilizarlos luegos
